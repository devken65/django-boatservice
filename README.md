# 웹사이트 기반 낚시배 예약 사이트
### 소개
이 프로젝트는 Django RestFramework(DRF)를 학습/응용을 목적으로 만든 개인 프로젝트입니다.  
전체적인 학습사항 
- Django
  - APP / Model 의 개념
  - Model의 Field 구축
  - Model과 DB 간의 소통 (ORM)
  - Template를 이용한 View구현
  - Django RestFramework란?
  - ViewSet 구축 (APIView)
  - 데이터를 검증 및 JSON 변환시켜주는 Serializer
  - ...

- common
  - APP 들이 사용하는 공통코드를 작성한 템플릿
- config
  - APP URL 등록 및 전체환경설정
- APPS
  - DB에 등록되는 실제 데이터 모음
